## Quick commands

Wrangler helper commands

```cmd
# development
yarn dev

# Build
yarn build

# build & deploy / publish
yarn deploy
```

Verify using curl commands

```cmd
curl -H "x-host: www.example.com" -H "accept: text/html" https://cloudflare-worker-demo.samar-2020.workers.dev | less
```
